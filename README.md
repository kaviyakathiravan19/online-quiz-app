# Online Quiz App

A colorful frontend-only Online Quiz Application built using HTML, CSS, and JavaScript.

## Features
- Admin & User login
- Multiple choice quiz
- Timer and score calculation
- Leaderboard
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage
